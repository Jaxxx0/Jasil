    Introduction and Pre-Security¶

    Level 1 - Intro¶

    • OpenVPN - 了解如何使用 OpenVPN 连接到虚拟专用网络.
    • Welcome - 了解如何使用 TryHackMe 房间开始您的网络安全技能提升.
    • Intro to Researching - 渗透测试研究技能简介.
    • Linux Fundamentals 1 - 踏上学习 Linux 基础知识的旅程. 学习在交互式终端上运行一些最基本的命令.
    • Linux Fundamentals 2 - 踏上学习 Linux 基础知识的旅程. 学习在交互式终端上运行一些最基本的命令.
    • Linux Fundamentals 3 - 踏上学习 Linux 基础知识的旅程. 学习在交互式终端上运行一些最基本的命令.
    • Pentesting fundamentals - 渗透测试的基础知识.
    • Principles of security - 安全原则.
    • Red Team Engagements - 介绍红队参与.
    • Hip Flask - 深入演练，涵盖针对易受攻击服务器的渗透测试方法.

    入门 CTF 入门

    • Osint - 开源情报简介.
    • Shodan.io - 了解 Shodan.io 以及如何使用它进行设备枚举.

    Free Beginner Red Team Path¶

    Level 2 - Tooling¶

    • Tmux - 学习使用 tmux，它是 Linux 上最强大的多任务处理工具之一.
    • Nmap - 体验强大的网络扫描工具 Nmap.
    • Web Scanning - 了解自动网页扫描的基础知识.
    • Sublist3r - 了解如何使用 Sublist3r 查找子域.
    • Metasploit - 介绍 Metasploit 框架的主要组件.
    • Hydra - 了解并使用 Hydra，一种快速的网络登录破解程序，以暴力破解并获取网站的凭据.
    • Linux Privesc - 在故意错误配置的 Debian VM 上练习您的 Linux 权限提升技能，并通过多种方式获得 root 权限！ SSH可用.
    • Red Team Fundamentals - 了解红色参与的基础知识、涉及的主要组成部分和利益相关者，以及红色团队与其他网络安全参与的不同之处.
    • Red Team Recon - 了解如何使用 DNS、高级搜索、Recon-ng 和 Maltego 来收集有关目标的信息.

    红队介绍 CTF

    • Vulnversity - 了解主动侦察、Web 应用程序攻击和权限升级.
    • Blue - 利用常见的错误配置问题部署和入侵 Windows 机器.
    • Simple CTF - 初级 CTF.
    • Bounty Hacker - 以太空牛仔为主题的启动到 root 机器.

    Level 3 - Crypto & Hashes with CTF practice¶

    • Crack the hash - 破解哈希挑战.
    • Agent Sudo - 你在深海下发现了一个秘密服务器. 你的任务是侵入服务器内部并揭露真相.
    • The Cod Caper - 指导室将带您渗透和利用 Linux 系统.
    • Ice - 部署并入侵 Windows 机器，利用安全性极差的媒体服务器.
    • Lazy Admin - 简单的 linux 机器来练习你的技能.
    • Basic Pentesting - 这是一台允许您练习网络应用程序黑客攻击和权限升级的机器.
    • Bypassing UAC - 了解绕过 Windows 主机中的用户帐户控制 (UAC) 的常用方法.

    Level 4 - Web¶

    • OWASP top 10 - 了解并利用每个 OWASP 十大漏洞； 10 大最严重的网络安全风险.
    • Inclusion - 初学者级别的 LFI 挑战.
    • Injection - 操作系统命令注入演练. 演示操作系统命令注入并解释如何在您的服务器上防止它.
    • Juiceshop - 这个房间使用 OWASP juice shop 易受攻击的 Web 应用程序来学习如何识别和利用常见的 Web 应用程序漏洞.
    • Overpass - 当一些破产的 CompSci 学生制作密码管理器时会发生什么.
    • Year of the Rabbit - 你能在不掉进洞里的情况下黑进兔年盒子吗？
    • DevelPy - 用于 FIT 的 Boot2root 机器并支持危地马拉 CTF.
    • Jack of all trades - Boot-to-root 最初是为 Securi-Tay 2020 设计的.
    • Bolt - 螺栓主题机器扎根.

    Level 5 - Reverse Engineering & Pwn¶

    • CC Radare2 - 这个房间教授 radare2 的基础知识.

    • Dumping Router Firmware - 逆向工程路由器固件.
    • Intro to pwntools - 流行的 pwn 工具框架介绍.

    Level 6 - PrivEsc¶

    • Sudo Security Bypass - 探索 Unix Sudo 程序中的 CVE-2019-14287 的教程室. SudoVulns 系列中的房间一.
    • Sudo Buffer Overflow - 探索 Unix Sudo 程序中的 CVE-2019-18634 的教程室. SudoVulns 系列中的二号房间.

    • Linux Privesc Arena - 学生将学习如何使用非常脆弱的 Linux VM 提升权限.
    • Windows Privesc - 学生将学习如何使用非常易受攻击的 Windows 7 虚拟机提升权限.
    • Blaster - Metasploit 框架获得立足点.
    • Ignite - 一家新成立的公司的网络服务器存在一些安全问题.
