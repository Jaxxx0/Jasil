    Awesome GameMaker ¶

    > GameMaker 的精彩库、片段、指南和项目的精选列表.

    GameMaker 是 YoYo Games 推出的一款用户友好的跨平台游戏引擎，它允许初学者和高级游戏开发人员为桌面、HTML5 和控制台平台创建 2D 和 3D 游戏.

    Getting Started¶

    • GameMaker Manual
    • GameMaker Release Notes
    • GameMaker Marketplace
    • Beginner GameMaker Tutorials - 肖恩·斯伯丁的教程. 全面介绍IDE的基本功能. 如果您是初学者，强烈建议您查看完整的播放列表.
    • Advanced GameMaker Tutorials - 来自 DragoniteSpam 的教程，深入探讨与 GML 语言相关的更高级主题. 他们还有全面的 3D 和着色器介绍视频.

    Recommendations¶

    • 如果您已经有编程经验，请学习 GameMaker 语言 (GML)，而不是视觉（拖放）功能.
    • 对于像素艺术， AESprite 是原生精灵编辑器的流行替代品.
    • 不要害怕使用其他开发者库. 其中很多都是免费的，这是有原因的. 只要注意许可证即可.
    • IDE 和运行时的更新可能会破坏您的游戏（例如 GML 的语法更改）. 如果您在一个团队中工作，请确保您运行的是相同版本的 GMS，并且仅在收到合理警告时才进行更新. 您可以在 GMS 下载页面重新安装以前版本的 IDE.
    • 除非您的游戏需要复杂的物理交互，否则通常建议避免使用 GameMaker 的内置物理系统.
    • 安排项目的例行备份. 如果您在存储库中处理较大的媒体文件，请尝试 Git LFS.

    Utilities¶

    • gm-core - 基础实用套件和新 GameMaker 项目的良好起点. 附带生活质量方法、网络工具、测试框架、增量计时等.
    • FAST - 灵活的助手工具包. 与 gm-core 类似，但具有输入和分辨率处理功能.
    • Stitch - 管道开发套件. 包括跨项目导入、批量创建/更新精灵和声音、纹理页面管理等等. 仅在 Windows 上测试.
    • DDDEditor - 通用游戏编辑器.
    • handytools - Juju 的库集合在一个方便的项目中.
    • GameMaker Scaffolding - 另一个很酷的包罗万象的模板，专注于构建低分辨率、基于图块的游戏.
    • Iota - Lightweight timestep library.
    • Stopwatch - GameMaker 警报更换.
    • wTimer - 强大的警报替代方案.
    • FrogAlarm - GameMaker 警报的另一个简单替代方案.
    • fuwafuwa - 易于使用的计时器系统.
    • Timer - 基于 JS 的 setTimeout 和 setInterval 的计时器方法.
    • Broadcast - 事件处理库.
    • EventGML - 轻量级且快速的 Node.js 风格事件库.
    • Polarca - 插值函数.
    • Twerp - 类似于 lerp() 的缓动函数.
    • Coroutines - GameMaker 的异步函数.
    • GML-OOP - 用于操作 GameMaker 主要功能的构造函数库.
    • GML-Classes - 另一个向 GameMaker 添加 OOP 功能的项目.
    • Map - 哈希表实现.
    • Matrices - 矩阵处理脚本的集合.
    • gm-stream - 数据结构操作.
    • Promises - JavaScript Promise 的改编版.
    • Destructors - 允许您在结构内部使用 ds_* 类型，例如列表和映射.
    • SNAP - 轻松的数据格式保存和加载. 请注意，较新版本的 GameMaker 包含“json_parse”和“json_stringify”. 但是，如果您要转换 csv、ini、xml 等，您可能会发现这非常有用.
    • Dynamo - 动态数据加载器.
    • LWO - 使用结构的轻量级对象.
    • Gumshoe - 简单的深度文件搜索功能.
    • Lock And Key - 字符串和文件加密.
    • Mathematical Scripts - 数学脚本的集合.
    • Seedpod - 用于改善 GML 编程体验的脚本集合.
    • Trixscript - 通过有用的功能为您的游戏注入活力.
    • CoreExtension - CC0 编程库的集合.
    • Voxeledphoton's FreeGMScripts - 附加 GML 辅助函数. 有些可能对于 2.3+ 语法来说已经过时了.
    • ForEach - 为数组、ds_lists、ds_maps、ds_stacks、ds_queues、ds_priorities 和结构添加 foreach 实现.
    • DeepCopy - 深度克隆类实例/构造结构、匿名结构和以任意顺序嵌套的数组！
    • Motion Scripts - 提供内置运动变量的替换方法.
    • Cottonwool - 安全表面，无内存泄漏.
    • Canvas - 另一个出色的表面管理解决方案.
    • zlib functions - 简单的压缩/解压功能.
    • Window Taskbar - 仅限Windows. 闪烁游戏窗口边框和/或其任务栏按钮.
    • GMSDLL - 用于为 GameMaker 构建 DLL 的模板项目.
    • Extension Collection - 一套各种扩展.
    • GMLodash - GML 中的函数式编程.
    • Wraptor - 包装函数库.
    • Autoframer - 自动处理在不同的显示器和窗口尺寸上调整游戏视图的大小.
    • gameframe - Windows 的自定义窗口标题和边框.
    • GML+ - 旨在“填补 GML 空白”的脚本集合.
    • GMLive - 实时编码/交互式编程.
    • GMEdit - 与 GameMaker 结合使用的代码编辑器.
    • YYP Maker - 为您制作“.yyp”文件.
    • Rubber - 通过命令行编译 GameMaker 项目. 这是一个 great guide 关于如何使用它.
    • gml-highscorer - 高分和奖杯系统.
    • SSave - 简单的文件保存系统.
    • GMD3D11 - 用于与 Windows Direct3D 连接的 DLL.
    • GMTimeLine - GameMaker 时间线的纯代码替代方案.
    • Catspeak - 用于模组支持的跨平台编程语言.
    • GMBenchmark - 一个对 GML 代码进行基准测试的工具.
    • Agenda - 安排和延迟回调的执行.

    Debugging¶

    • rt-shell - 易于使用的游戏外壳. 创建您自己的命令、命令元数据、命令建议、历史记录等.
    • Olympus - 测试框架.
    • Crispy - GameMaker 中的单元测试.
    • Snitch - 崩溃和日志系统.
    • DeerLog - 小日志作家.
    • gms2-test - 单元测试框架.
    • Meta - 运行时资产检查器.
    • FPS Speedometer - 漂亮的帧率显示.
    • Duck - 快速 GML 分析器，用于强制代码样式并检测错误.

    Input Handling¶

    • Input - 没有废话的游戏手柄/键盘库.
    • XeroInput - 另一个库可以处理单个操作的多个输入.
    • InputCandy - 与 Input 类似，它充当 SDL 的包装器，但也提供测试、屏幕诊断以及与外设相关的一些其他 UI 组件.
    • Good Vibes - 设备振动.
    • Mouse Queue - 高精度跟踪 Windows 鼠标指针.
    • Native Cursors - 系统级自定义光标.

    User Interface¶

    • Scribble - 高效的多效果文本渲染器.
    • Scripture - 另一种易于使用、高度兼容的文本渲染器.
    • Chatterbox - 叙事脚本工具.
    • Textboxy - 简单的文本框.
    • Crochet - 适合作家和程序员的交互式对话编辑器.
    • NotificationSystem - GameMaker 中的通知.
    • YUI - 具有实时重新加载、模板系统、数据绑定和拖放功能的 UI 系统.
    • Guido - 简单的即时模式 GUI 框架.
    • GMUI-Framework - 用于构建和控制菜单的纯 GML 解决方案，与 .NET UI 类似.
    • GMS2-UI-Library - 一个充满有用脚本的库，用于在 GameMaker 中实现 UI 设计.
    • Emu UI - 常见的 UI 元素（文本输入、复选框、单选按钮、对话框等）.
    • Easy And Fast Menu - 简单的实现即可在几秒钟内启动并运行菜单. 如果您不寻找像 GMUI 这样的更大解决方案，那么这似乎是一个非常合适的选择.
    • Pause Menu - 另一个较小的实现，但在菜单选项之间有一个很酷的动画.
    • Shampoo - 使用类似标记语言的 GUI.
    • Magpie - 通用库存系统.
    • ImGuiGML - Dear ImGui 的 DLL/GML 包装器.
    • GUI Framework - GUI implementation from Niris Games.
    • Menu Tutorial - 友好的宇航员.
    • Smart Clickable GUI - 像素化教皇.
    • RTS Selection Tool - 鼠标拖动功能可在实时策略游戏中选择棋子.
    • zitk - 另一个有趣的、受 Dear ImGui 启发的 GUI 工具包. 正在开发中，但值得关注.
    • gooey - 适用于 GameMaker LTS 的基于 Sprite 的 UI 库.

    Localization¶

    • polyglot - 本地化库.
    • gm-i18n - 使用 JSON 文件简单快速地进行文本国际化.
    • lexicon - 另一个专注于简化实施的本地化解决方案.
    • GMLocalize - 不是一个完整的本地化解决方案. 从 GameMaker Studio 2 项目中提取用于本地化的文本字符串并将其保存到 JSON 文件.
    • Small Pentapop Localization Tool - 与 GMLocalize 类似的导出工具，但导出为 csv.

    Physics¶

    • Loj Hadron Collider - 强大的、像素完美的碰撞引擎.
    • On Slopes and Grids - 实现 45° 斜坡的教程.
    • GMS2 Platforming System - 由 Ben Allen 实施的 GameMaker 以及 Shaun Spalding 原始 1.4 平台游戏教程的扩展.
    • Inverse Kinematics Extension - 用于处理逆运动学的库.
    • Verlet Integration Library - Sarek Lambert 的 Verlet 集成.
    • GMVerlet-Integration - 用于视觉效果的 Verlet 集成示例.
    • Top-Down Movement & Collision - 来自教皇的强大的基于对象的碰撞系统.

    Sprites¶

    • AESnips - 精灵播放系统.
    • phgen - 占位符资产生成.
    • Disarm - 运行时的精灵骨骼动画.
    • Spritely - 2D 视频游戏精灵的图像校正和清理.
    • PixelUpscaler - 像素艺术升级着色器，为 GameMaker 提供尴尬的分辨率.
    • ASESync - 自动同步 GameMaker 中的 aesprite 文件.
    • conveyorbelt - 与 ASESync 类似. 将 Aesprite 文件导出到 GameMaker sprite.
    • Collage - 纹理页面生成器和图像管理器. 模仿 GameMaker 的纹理页面打包，同时提供更高的灵活性.

    Audio¶

    • FML - FMOD Studio API 的 GameMaker 绑定.
    • FMODGMS - 这并不支持 FMOD 提供的所有功能，并且该项目本身已被搁置.
    • Echo/Delay Effect - 优化延迟效果.
    • wavload - 演示如何从外部加载 .wav 文件.
    • audioExt - 健全的外部装载机/卸载机管理器.
    • ExternalAudio - 在运行时加载外部 .wav 文件.
    • Phonix - 紧凑型音频系统. 非常适合动感音乐！
    • Vinyl - 实时更新音频系统.
    • LineAudio - 音频助手功能.
    • Bard - 用于在 GameMaker 中设计和实现良好音频的引擎. 已更新以利用更新的 GameMaker 音频效果.

    Levels¶

    • LDtk to GMS - LDtk 进口商.
    • LDtkParser - 高级 LDtk 导入器.
